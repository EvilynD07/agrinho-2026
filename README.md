# Agrinho 2026 — Campo e Cidade: Conexões que Transformam 🌾💜🏙️

Este repositório contém o código-fonte de uma aplicação web interativa desenvolvida especialmente para o **Concurso Agrinho 2026**. O projeto utiliza a tecnologia para contar a história da interdependência entre o meio rural (campo) e o meio urbano (cidade), destacando a inovação, a robótica e a sustentabilidade.

---

## 🎨 Identidade Visual e Conceito
A paleta de cores do projeto foi planejada para unir os dois universos do tema:
* **Roxo Tecnológico (`#6a1b9a`):** Representa a inovação, os softwares, a robótica e a inteligência desenvolvida nas cidades.
* **Verde Campo (`#2e7d32`):** Representa a sustentabilidade, a produção agrícola, o meio ambiente e a força do interior.

---

## 🚀 Novas Funcionalidades e Elementos
* **Seção de Pilares Interativos:** Uma grade (*Grid*) de cartões modernos que destacam o uso de Drones, Robótica e Gestão de Recursos no campo.
* **Quiz com Sistema de Pontuação:** Jogo interativo em JavaScript que valida a resposta do usuário em tempo real e gerencia uma pontuação dinâmica (ganho de pontos sem repetição acumulativa).
* **Design Responsivo Avançado:** Menus e seções que se reorganizam de forma elegante tanto em computadores quanto em telas de celulares.

---

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização baseada em variáveis de cor, layouts em Grid/Flexbox e efeitos de transição (*hover*).
* **JavaScript (Vanilla):** Lógica do motor do quiz, controle de estados de clique e manipulação dinâmica do DOM.
