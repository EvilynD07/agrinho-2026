# Projeto Agrinho 2026 🌾🏙️
> **Tema:** Campo e Cidade: Conexões que transformam

Este é um projeto web interativo desenvolvido para o Concurso Agrinho 2026. A aplicação explora a interdependência entre o meio rural e o urbano, destacando como a tecnologia e a sustentabilidade unem esses dois mundos para construir um futuro melhor.

## 🚀 Funcionalidades
* **Design Responsivo:** A interface se adapta automaticamente a computadores, tablets e celulares.
* **Menu de Navegação:** Âncoras suaves para navegar entre as seções da página.
* **Quiz Interativo:** Um jogo de perguntas e respostas dinâmico feito em JavaScript para testar os conhecimentos do usuário sobre o agronegócio tecnológico e sustentável.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização moderna baseada em variáveis, Grid/Flexbox e efeitos de transição.
* **JavaScript (Vanilla):** Lógica do quiz e manipulação do DOM em tempo real.
